# House-Prediction-Analysis
A data analysis and visualization project using Python and Power BI to explore factors affecting house prices.
# House-Prediction-Analysis-Dashboard-Powerbi-Python

Project Overview
This project analyzes housing data to understand the factors affecting house prices using data analysis and visualization techniques.

---

## Objectives
- Analyze relationships between house price and features like rooms, crime rate, and location
- Build a classification model to categorize house prices
- Create an interactive dashboard using Power BI

---

## Tools Used
- Python (Pandas, Scikit-learn)
- Power BI
- Excel

---

## Key Insights
- Most houses are far from river
- Higher crime rate areas tend to have lower house prices
- Majority of houses fall into the low-price category
